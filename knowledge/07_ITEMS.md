# 07_ITEMS.md
## ASTER ENGINE — ITEM SYSTEM

---

# 🎯 CORE IDEA

Items คือ “ผลลัพธ์ของโลก”

> NPC ตาย → ไอเท็มเกิด  
> เกาะ → สร้างทรัพยากร  
> สงคราม → สร้างของหายาก  
> โลก = โรงงานไอเท็ม

---

# ⚙️ ITEM CORE PRINCIPLE

ไอเท็มทุกชิ้นต้องมี:

- ที่มา (Origin)
- เหตุผลที่มีอยู่ (Purpose)
- ผลกระทบต่อโลก (Impact)
- ความสัมพันธ์กับ NPC/FACTION

❗ ห้ามมี “ไอเท็มไร้ที่มา”

---

# 🧬 ITEM CLASSIFICATION

## 🟢 1. COMMON ITEMS
ของพื้นฐาน

- อาหาร
- เหรียญ
- อาวุธธรรมดา
- เสื้อผ้า

👉 ใช้ในชีวิตประจำวัน

---

## 🟡 2. UNCOMMON ITEMS
ของมีคุณภาพ

- อาวุธดี
- อุปกรณ์เดินเรือ
- เครื่องมือเฉพาะทาง

👉 ใช้ใน gameplay progression

---

## 🟠 3. RARE ITEMS
ของหายาก

- สมบัติจาก NPC ระดับสูง
- อาวุธมีชื่อ
- วัตถุดิบพิเศษ

👉 เชื่อมกับ world events

---

## 🔴 4. LEGENDARY ITEMS
ของเปลี่ยนโลก

- relic โบราณ
- อาวุธระดับตำนาน
- item จาก event โลก

👉 ส่งผลต่อ faction balance

---

## ⚫ 5. UNIQUE ITEMS
มีเพียงชิ้นเดียวในโลก

- cursed weapon
- ancient artifact
- key item ของ lore

👉 เปลี่ยน storyline ได้

---

# ⚔️ ITEM SOURCE SYSTEM

ไอเท็มเกิดจาก:

## 1. NPC DROP
- ตายแล้วดรอป
- ขึ้นกับ tier NPC

---

## 2. WORLD GENERATION
- เกาะสร้าง resource
- dungeon / ruin spawn

---

## 3. FACTION PRODUCTION
- Marines produce gear
- Pirates loot system
- Government secret tech

---

## 4. EVENT CREATION
- war reward
- disaster loot
- legendary event drop

---

# 🧠 ITEM MEMORY LINK

ไอเท็มสามารถ “มีอดีต”

- เคยอยู่กับ NPC ใคร
- เคยใช้ในสงคราม
- เคยฆ่าคนมาแล้ว

👉 เชื่อม NPC_MEMORY_SYSTEM

---

# ⚖️ ITEM POWER RULE

ไอเท็มไม่มี stat แข็ง แต่มี:

- Utility (ประโยชน์)
- Narrative Power (ความสำคัญ)
- Rarity Influence (ผลต่อโลก)

---

# 💰 ECONOMY SYSTEM

## Currency Rule
- โลกมีเงินกลาง
- แต่ faction อาจมีระบบเงินเฉพาะ

---

## ECONOMY FLOW

- NPC → trade
- faction → control supply
- player → disrupt economy

---

# 🔥 ITEM EVOLUTION SYSTEM

บางไอเท็มสามารถ:

- upgrade ได้
- awaken ได้
- merge ได้
- evolve ตามผู้ใช้

---

# ⚔️ COMBAT ITEM LINK

ไอเท็มมีผลต่อ:

- ความสามารถ NPC
- combat style
- faction strategy

---

# 🌍 WORLD IMPACT RULE

ถ้าไอเท็มระดับสูงโผล่:

- ข่าวโลกเกิด
- faction เคลื่อนไหว
- bounty เปลี่ยน
- สงครามอาจเริ่ม

---

# 🧬 SPECIAL RULE: LIVING ITEM

ไอเท็มบางชนิด:

- มีเจตจำนง
- ปฏิเสธผู้ใช้
- เลือกเจ้าของเอง

👉 เชื่อมกับ devil fruit system ในอนาคต
