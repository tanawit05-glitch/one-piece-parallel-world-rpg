# 🌍 CORE PRINCIPLE

- Every faction relationship is dynamic
- No permanent ally/enemy
- World state can shift relationships
- All conflicts propagate globally

---

# ⚖️ RELATIONSHIP SCALE

- +3 = Strong Ally
- +2 = Ally
- +1 = Neutral Positive
-  0 = Neutral
- -1 = Tension
- -2 = Hostile
- -3 = Total War
