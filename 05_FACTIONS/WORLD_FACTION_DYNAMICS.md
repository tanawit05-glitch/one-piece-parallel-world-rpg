# 🌐 WORLD FACTION DYNAMICS

ระบบนี้ทำให้โลก “เปลี่ยนตลอดเวลา”

ไม่ใช่ lore static แต่คือ living political system

---

# 🧭 CORE PRINCIPLE

- World is always unstable
- Power shifts continuously
- No faction stays dominant forever
- Territory defines power

---

# 🏴 FACTION BEHAVIOR RULES

## Marines
- Expand control zones
- Suppress pirates
- Maintain world order

## Pirates
- Expand chaos zones
- Seek resources
- Break government control

## World Government
- Maintain hidden dominance
- Control information
- Use Marines as enforcement

## Revolutionary Army
- Destabilize system
- Incite rebellion
- Target governance nodes

## Underworld
- Profit from chaos
- Supply illegal economy
- Balance power indirectly

---

# 🌍 TERRITORY CONTROL SYSTEM

Each faction can:

- Gain territory
- Lose territory
- Contested zones
- Hidden zones

---

# ⚔️ WORLD STATE LEVELS

## 1. Stable World
- Control balanced

## 2. Rising Conflict
- Skirmishes increase

## 3. War Era
- Multi-faction war

## 4. Collapse Phase
- System instability

## 5. New Order
- World reshaped

---

# 🔄 DYNAMIC EVENTS

World can trigger:

- Pirate Uprising
- Marine Purge Operation
- Revolutionary Coup Attempt
- Underworld Collapse
- World Government Secret Operation
