# 🌍 FACTION SYSTEM OVERVIEW


FACTION DATABASE คือฐานข้อมูลกลางขององค์กรและกลุ่มอำนาจทั้งหมดในโลก


Faction ไม่ใช่เพียงรายชื่อองค์กร

แต่คือ:

- กลุ่มคน
- อุดมการณ์
- อำนาจ
- ทรัพยากร
- อิทธิพล
- ประวัติศาสตร์


ทุก Faction สามารถ:

- เติบโต
- เสื่อมถอย
- แตกแยก
- เปลี่ยนผู้นำ
- สูญเสียอำนาจ
- กลายเป็นตำนาน
