# 🧭 CORE PRINCIPLE

- Events drive the world
- No event is isolated
- Every conflict has ripple effects
- Story is generated, not written

---

# 🔥 EVENT CATEGORIES

## 1. Military Events
- War outbreak
- Fleet battle
- Island invasion

## 2. Political Events
- Coup d’état
- Treaty breakdown
- Government purge

## 3. Criminal Events
- Pirate uprising
- Underworld war
- Bounty escalation wave

## 4. World Events
- Natural disaster
- Sea anomaly
- Ancient weapon activation

---

# ⚙️ EVENT GENERATION RULE

Every event must have:

- Trigger cause
- Involved factions
- Location
- Consequence chain

---

# 🌊 EVENT PROPAGATION SYSTEM

One event affects:

- Territory
- Bounty System
- Ship routes
- Faction relations
- NPC behavior

---

# ⚔️ EVENT INTENSITY LEVELS

- Level 1: Local incident
- Level 2: Regional conflict
- Level 3: Multi-island war
- Level 4: Faction war
- Level 5: World-scale crisis

---

# 🧠 EXAMPLE EVENT

## “Iron Tide Incident”

- Trigger: Pirate raid on Marine supply line
- Factions: Pirates vs Marines
- Location: East Blue route
- Consequence:
  - Marine patrol increase
  - Pirate bounty spike
  - Underworld weapon trade increases

---

# ⚠️ RULE OF CONSEQUENCE

No event can end without:

- Change in world state
- Change in faction relation
- Change in territory OR influence
