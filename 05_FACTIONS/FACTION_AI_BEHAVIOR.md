# ⚖️ SYSTEM TYPE

## HYBRID INTELLIGENCE MODEL

ผสม 3 อย่าง:

- 🎯 Tactical Logic (เหตุผลเชิงกลยุทธ์)
- 🌊 Narrative Instinct (แรงเล่าเรื่อง)
- ⚔️ Reactive Behavior (ตอบสนองเหตุการณ์)

---

# 🧭 CORE PRINCIPLE

- Faction is not static
- Faction is an evolving intelligence
- Decisions are probabilistic, not fixed
- World events reshape decision logic

---

# 🧠 FACTION THINKING LOOP

ทุก faction จะทำ loop นี้ตลอดเวลา:

## 1. PERCEPTION (รับรู้โลก)
- ตรวจสอบ territory
- ตรวจสอบภัยคุกคาม
- ตรวจสอบโอกาส

## 2. EVALUATION (ประเมิน)
- Threat Level
- Opportunity Level
- Resource Status
- Narrative Pressure

## 3. DECISION (ตัดสินใจ)
- เลือก action ที่ดีที่สุดตาม priority

## 4. EXECUTION (ลงมือ)
- ส่ง fleet
- เริ่ม war
- เจรจา
- ถอนกำลัง

## 5. MEMORY UPDATE (จดจำ)
- บันทึกผลลัพธ์
- อัปเดตความสัมพันธ์

## 🟦 MARINES

## 🏴 PIRATES
- Risk: High

- Logic: Gain-first thinking

## 🏛 WORLD GOVERNMENT
- Risk: Low (public)
- Behavior: Hidden Control
- Logic: Stability over truth

## 🔥 REVOLUTIONARY ARMY
- Risk: High
- Behavior: Strategic Disruption
- Logic: System collapse optimization

## 🕶 UNDERWORLD
- Risk: Adaptive
- Behavior: Profit Maximization
- Logic: Balance exploitation

# ⚖️ DECISION WEIGHT SYSTEM

ทุกการตัดสินใจใช้ “น้ำหนัก”:

- Survival Priority (40%)
- Expansion Priority (30%)
- Revenge / History (10%)
- Opportunity (20%)

👉 แต่ weight เปลี่ยนตามสถานการณ์โลก

# 🌊 NARRATIVE OVERRIDE SYSTEM

- Emotional world pressure
