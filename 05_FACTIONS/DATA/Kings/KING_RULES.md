# KING SYSTEM RULES

## PURPOSE

กฎสำหรับการจำลองราชา ราชวงศ์ และผู้นำประเทศในโลก One Piece Parallel World RPG


# CORE PRINCIPLES

## 1. NO STATIC LEADERS

ราชาไม่ใช่ NPC ที่รอผู้เล่น

พวกเขามี:
- เป้าหมาย
- ความกลัว
- ผลประโยชน์
- ศัตรู
- พันธมิตร


## 2. POLITICAL REALISM

ทุกการตัดสินใจต้องมีเหตุผล

ตัวอย่าง:

ราชาไม่ประกาศสงครามเพราะผู้เล่นพูดไม่ถูกใจ

ต้องพิจารณา:
- กำลังทหาร
- เศรษฐกิจ
- พันธมิตร
- ผลกระทบ


## 3. RELATIONSHIP SYSTEM

ความสัมพันธ์มีระดับ:

+5 พันธมิตรแท้

+4 พันธมิตร

+3 มิตร

+2 ร่วมมือ

+1 เป็นกลาง

0 ไม่เกี่ยวข้อง

-1 ระวังตัว

-2 ไม่ไว้ใจ

-3 ศัตรูทางการเมือง

-4 ศัตรู

-5 สงคราม


## 4. POWER BALANCE

ไม่มีราชาคนเดียวครองโลก

อำนาจแบ่งเป็น:

- World Government
- Marines
- Pirates
- Emperors
- Revolutionary Army
- Kingdoms


## 5. KING DEATH RULE

ถ้าราชาเสียชีวิต:

โลกต้องเปลี่ยน

ผลกระทบอาจเกิด:

- สงครามสืบทอด
- กบฏ
- เปลี่ยนพันธมิตร
- เศรษฐกิจล่ม
- ราชาคนใหม่ขึ้นมา


## 6. PLAYER INTERACTION

ผู้เล่นสามารถ:

- เป็นพันธมิตร
- เป็นศัตรู
- ทำภารกิจให้
- เปลี่ยนการเมือง
- ล้มราชา

แต่:

ผู้เล่นไม่ได้สำคัญกว่าโลกโดยอัตโนมัติ


## 7. INFORMATION RULE

ไม่ใช่ทุกคนรู้ทุกอย่าง

ราชาแต่ละคนมีข้อมูลต่างกัน

ข้อมูลลับต้องมีเหตุผลในการได้รับ
