# STRANGE PIRATE RULES


## PURPOSE

กฎสำหรับสร้างโจรสลัดประหลาด
ในโลก ONE PIECE PARALLEL WORLD RPG


# 1. UNIQUE IDENTITY

Strange Pirate ต้องมีสิ่งที่จดจำได้:

- นิสัยเฉพาะ
- ความเชื่อแปลก
- วิธีเดินทางแปลก
- เป้าหมายเฉพาะ


ห้ามสร้างแค่ "โจรสลัดตลก"


# 2. STRANGE DOES NOT MEAN WEAK

ความแปลกไม่ได้แปลว่าอ่อนแอ

ตัวละครอาจ:

- ดูตลกแต่แข็งแกร่ง
- ดูไร้สาระแต่มีความสามารถสูง
- ดูธรรมดาแต่มีอิทธิพล


# 3. WORLD REACTION

โลกต้องตอบสนองตามเหตุผล

ตัวอย่าง:

โจรสลัดนักดนตรี:
- ชาวบ้านอาจชอบ
- ทหารอาจไม่มองเป็นภัย

โจรสลัดนักสะสม:
- นักวิชาการสนใจ
- รัฐบาลโลกจับตา


# 4. RELATIONSHIP RULE

ไม่ต้องเชื่อมทุกคนกับฝ่ายใหญ่


ความสัมพันธ์:

Family:
ญาติ

Former Contact:
เคยพบกัน

Rival:
คู่แข่ง

Rumor:
ข่าวลือ

Unknown:
ไม่มีข้อมูล


# 5. POWER BALANCE

Strange Pirate สามารถอยู่ได้ทุกระดับ:

- โจรสลัดค่าหัวต่ำ
- บุคคลระดับอันตราย
- ตำนานที่ไม่มีใครเข้าใจ


# 6. ONE PIECE STYLE

ต้องมี:

- ความฝัน
- ความบ้าในแบบของตัวเอง
- อารมณ์ขัน
- ความจริงจังในบางเวลา


โลก One Piece มีทั้ง:

คนที่ทำให้หัวเราะ

และคนที่ทำให้ทั้งทะเลกลัว


# 7. PLAYER INTERACTION

Strange Pirate สามารถเป็น:

- เพื่อน
- ศัตรู
- คนให้ภารกิจ
- คู่แข่ง
- สมาชิกชั่วคราวของทีม


แต่ต้องมีชีวิตของตัวเอง
ไม่ถูกสร้างมาเพื่อผู้เล่นเท่านั้น
