# STRANGE_PIRATE_001

## BASIC INFORMATION

- Crew Name: Floating Inn Pirates
- Captain: บาร์โต้ ลูเมียร์ (Barto Lumiere)
- Title: เจ้าของโรงแรมกลางทะเล
- Origin: Grand Line
- Bounty: 88,000,000 Berries
- Type: Strange Pirate Crew


## PERSONALITY

ใจดี พูดมาก ชอบต้อนรับคนแปลกหน้า

นิสัยแปลก:
เขาจะถามชื่อแขกทุกคนก่อนถามว่าเป็นศัตรูหรือไม่


## HISTORY

บาร์โต้เปลี่ยนเรือโจรสลัดเก่าให้กลายเป็นโรงแรมลอยน้ำ

เขารับทุกคน:
- พ่อค้า
- นักเดินทาง
- โจรสลัด
- ทหารที่หลงทาง

แต่ถ้ามีใครสร้างปัญหาในโรงแรม
เขาจะจัดการทันที


## STYLE

- ไม้เท้า
- การต่อสู้ป้องกัน


## CONNECTIONS

- เคยให้ที่พักกับ PIRATE_ROOKIE_001 เรย์ โฮชิโนะ

- ไม่มีความสัมพันธ์กับจักรพรรดิ
