# SOLO PIRATE RULES


## PURPOSE

กฎสำหรับสร้างและจำลองโจรสลัดเดี่ยว


# 1. INDIVIDUAL IDENTITY

Solo Pirate ต้องมี:

- ความฝัน
- เหตุผลที่เดินทางคนเดียว
- จุดเด่นเฉพาะตัว
- วิธีต่อสู้ของตัวเอง


ไม่ใช่แค่ "กัปตันที่ไม่มีลูกเรือ"


# 2. SOLO DOES NOT MEAN WEAK

การไม่มี Crew ไม่ได้หมายความว่าอ่อนแอ


Solo Pirate สามารถ:

- เป็นนักสู้ระดับสูง
- มีชื่อเสียงมากกว่า Crew ใหญ่
- เป็นตำนานของบางพื้นที่


# 3. REASON FOR BEING SOLO

ต้องมีเหตุผล เช่น:

- ไม่ไว้ใจคน
- สูญเสีย Crew
- ต้องการอิสระ
- กำลังค้นหาบางอย่าง
- มีนิสัยเข้ากับคนยาก


# 4. RELATIONSHIP SYSTEM

ความสัมพันธ์:

## Family

สายเลือดหรือญาติ


## Former Crew

อดีตลูกเรือ


## Rival

คู่แข่ง


## Mentor

อาจารย์หรือผู้ฝึกสอน


## Rumor

ข่าวลือ


## Unknown

ไม่มีข้อมูล


# 5. WORLD SIMULATION

Solo Pirate สามารถ:

- สร้าง Crew ใหม่
- เข้าร่วม Crew อื่น
- ถูกจับ
- หายตัว
- กลายเป็นตำนาน


โลกไม่หยุดรอพวกเขา


# 6. PLAYER INTERACTION

ผู้เล่นสามารถ:

- ท้าดวล
- ขอความช่วยเหลือ
- รับเป็นพันธมิตร
- เรียนรู้จากพวกเขา


แต่ Solo Pirate มีเป้าหมายของตัวเอง


# 7. ONE PIECE STYLE

Solo Pirate ควรมี:

- บุคลิกจำง่าย
- ความฝันชัดเจน
- ความแปลกเฉพาะตัว
- เรื่องราวที่ทำให้คนจำได้


บางครั้งคนคนเดียว...
