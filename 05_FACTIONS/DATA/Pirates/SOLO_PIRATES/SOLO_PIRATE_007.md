# SOLO_PIRATE_007

## BASIC INFORMATION

- Name: มิโอะ คุซานางิ (Mio Kusanagi)
- Title: นักล่าผีแห่งทะเล
- Origin: Grand Line
- Bounty: 240,000,000 Berries
- Type: Solo Pirate


## PROFILE

หญิงสาวที่เดินทางตามล่าเรือผีและเรื่องลึกลับในทะเล

หลายคนคิดว่าเธอเป็นนักล่าสมบัติ
แต่จริง ๆ เธอตามหาความจริงเกี่ยวกับอดีตครอบครัว


## STYLE

- มีดสั้น
- กับดัก
- การสืบสวน


## CONNECTIONS

- มีนามสกุลเดียวกับตระกูลนักดาบเก่าบางสาย แต่ไม่เกี่ยวข้องกับ KING_007
- เคยร่วมสำรวจซากเรือกับ PIRATE_010 นาโอมิ อาซากุระ
- เคยถูกเข้าใจผิดว่าเป็นสมาชิก Underworld
