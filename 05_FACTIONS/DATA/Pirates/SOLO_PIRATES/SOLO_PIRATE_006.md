# SOLO_PIRATE_006

## BASIC INFORMATION

- Name: เร็นโซ่ วาเลน (Renzo Valen)
- Title: หมอเถื่อนแห่งทะเล
- Origin: West Blue
- Bounty: 185,000,000 Berries
- Type: Solo Pirate


## PROFILE

อดีตแพทย์ทหารที่ออกจากกองทัพหลังเห็นสงครามมากเกินไป

ปัจจุบันรักษาทุกคนที่จ่ายไหว
ไม่สนว่าเป็นโจรสลัด ทหาร หรือพลเรือน


## STYLE

- การแพทย์สนาม
- อาวุธเคมี
- การต่อสู้ด้วยมีดผ่าตัด


## CONNECTIONS

- เคยศึกษาการแพทย์กับกลุ่มของ PIRATE_001 ลูเซียโน่ เวโรนา
- ราชินีลิอาน่า อาร์เวล (KING_006) เคยเชิญให้เข้าร่วมโครงการแพทย์ของอาณาจักร
- ไม่ชอบรัฐบาลโลก เพราะมองว่าพวกเขาจำกัดการรักษา
