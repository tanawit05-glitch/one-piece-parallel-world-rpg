# GREAT PIRATE CREW RULES


## PURPOSE

กฎสำหรับจำลองโจรสลัดระดับสูงในโลก
ONE PIECE PARALLEL WORLD RPG


# 1. CREW IDENTITY

ทุกกลุ่มต้องมี:

- เป้าหมาย
- สไตล์การเดินเรือ
- วิธีหาเงิน
- วิธีต่อสู้
- ความเชื่อ


โจรสลัดแต่ละกลุ่มต้องแตกต่างกัน


# 2. POWER BALANCE

Great Pirate Crew อยู่ระหว่าง:

Rookie Pirates
↓
Great Pirate Crew
↓
Emperor Crew


ไม่ควรมีพลังเทียบจักรพรรดิตั้งแต่เริ่ม


# 3. BOUNTY RULE

ค่าหัวสะท้อน:

- ภัยต่อรัฐบาลโลก
- อิทธิพล
- จำนวนเหตุการณ์
- ความเสียหาย


ไม่ใช่พลังต่อสู้อย่างเดียว


# 4. WORLD CONNECTION

ไม่จำเป็นต้องรู้จักทุกฝ่าย

ความสัมพันธ์แบ่งเป็น:

- Direct Connection
รู้จักหรือเคยพบกัน

- Indirect Connection
มีข่าวหรือผลกระทบเกี่ยวข้อง

- Unknown
ไม่มีข้อมูล


# 5. CREW DEVELOPMENT

กลุ่มโจรสลัดสามารถ:

- เติบโต
- แตกแยก
- ถูกจับ
- เปลี่ยนกัปตัน
- กลายเป็นจักรพรรดิในอนาคต


# 6. PLAYER INTERACTION

ผู้เล่นสามารถ:

- ร่วมมือ
- แข่งขัน
- ทำลาย
- เข้าร่วม
- สร้างชื่อเหนือพวกเขา


แต่ Great Pirate Crew ไม่หยุดรอผู้เล่น

โลกเดินต่อเสมอ


# 7. ONE PIECE STYLE

โจรสลัดควรมี:

- ความฝัน
- ความแปลก
- บุคลิกเฉพาะ
- ลูกเรือที่มีเอกลักษณ์

ไม่ใช่แค่กลุ่มนักรบ
