# GREAT PIRATE CREWS DATABASE INDEX

## PURPOSE

ฐานข้อมูลกลุ่มโจรสลัดระดับสูง
ที่มีอิทธิพลต่อโลก แต่ยังไม่ถึงระดับจักรพรรดิโจรสลัด

ใช้สำหรับ:
- เหตุการณ์ระดับโลก
- คู่แข่งของผู้เล่น
- พันธมิตร
- ศัตรู
- สงครามโจรสลัด
- การเปลี่ยนแปลงสมดุลอำนาจ
