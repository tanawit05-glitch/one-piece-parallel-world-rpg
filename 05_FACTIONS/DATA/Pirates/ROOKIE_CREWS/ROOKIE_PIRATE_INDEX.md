# ROOKIE LEVEL

Rookie Pirates แบ่งเป็น:

## Beginner Rookie

ค่าหัว:
ต่ำกว่า 50M

ลักษณะ:
- กลุ่มเล็ก
- ยังไม่มีชื่อเสียง
- มักพบใน Blue Sea


## Rising Rookie

ค่าหัว:
50M - 150M

ลักษณะ:
- เริ่มมีชื่อเสียง
- มีลูกเรือที่มีความสามารถ
- เริ่มถูกจับตามอง


## Dangerous Rookie

ค่าหัว:
150M - 300M

ลักษณะ:
- มีโอกาสเติบโตเป็น Great Pirate
- เริ่มมีอิทธิพลต่อพื้นที่
