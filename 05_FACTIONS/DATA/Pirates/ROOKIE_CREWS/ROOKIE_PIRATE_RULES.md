# ROOKIE PIRATE CREW RULES


## PURPOSE

กฎสำหรับจำลองโจรสลัดรุ่นใหม่
ในโลก ONE PIECE PARALLEL WORLD RPG


# 1. IDENTITY RULE

โจรสลัดหน้าใหม่ทุกกลุ่มต้องมี:

- ความฝัน
- เหตุผลในการออกทะเล
- บุคลิกเฉพาะ
- รูปแบบลูกเรือ
- จุดอ่อน


ไม่สร้างกลุ่มที่เป็นแค่ศัตรูสุ่ม


# 2. POWER RULE

Rookie Pirate ไม่ควรเริ่มต้นด้วยพลังระดับจักรพรรดิ


ระดับ:

Rookie
↓
Rising Rookie
↓
Great Pirate
↓
Emperor Candidate


การเติบโตต้องเกิดจาก:

- เหตุการณ์
- ชัยชนะ
- ชื่อเสียง
- พันธมิตร
- การเปลี่ยนแปลงของโลก


# 3. BOUNTY RULE

ค่าหัวไม่ได้วัดพลังต่อสู้อย่างเดียว


ค่าหัวขึ้นจาก:

- ทำลายทรัพย์สินรัฐบาล
- เป็นภัยต่อประชาชน
- มีอิทธิพล
- มีข้อมูลสำคัญ
- เอาชนะบุคคลสำคัญ


# 4. RELATIONSHIP RULE

Rookie Pirates ไม่จำเป็นต้องรู้จักทุกฝ่าย


ประเภทความสัมพันธ์:

Direct:
เคยพบกัน

Indirect:
เคยได้รับผลกระทบ

Rumor:
แค่มีข่าวลือ

Unknown:
ไม่มีข้อมูล


# 5. WORLD SIMULATION

Rookie Pirates สามารถ:

- โตขึ้น
- ถูกจับ
- แตกกลุ่ม
- เปลี่ยนกัปตัน
- เป็นพันธมิตร
- กลายเป็นศัตรู


โลกต้องเดินต่อแม้ผู้เล่นไม่เกี่ยวข้อง


# 6. PLAYER INTERACTION

ผู้เล่นสามารถ:

- แข่งขัน
- ร่วมมือ
- รับเข้ากลุ่ม
- แย่งชิงชื่อเสียง
- ช่วยให้เติบโต


แต่ Rookie Pirate ไม่ได้ถูกสร้างมาเพื่อแพ้ผู้เล่นเสมอ


# 7. ONE PIECE STYLE

Rookie Pirates ควรมี:

- ความฝันที่ชัดเจน
- ความแปลกเฉพาะตัว
- ลูกเรือที่มีสีสัน
- เรื่องราวส่วนตัว


โจรสลัดที่ค่าหัวต่ำก็สามารถสร้างตำนานได้
