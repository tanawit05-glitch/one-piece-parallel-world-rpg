# Underworld

## Overview

ฐานข้อมูลโลกใต้ดินของโลกคู่ขนาน

เครือข่ายอาชญากรรมและธุรกิจลับที่อยู่เบื้องหลังเหตุการณ์สำคัญของโลก

Underworld ไม่ใช่องค์กรเดียว แต่เป็นระบบผลประโยชน์ที่เชื่อมโยงระหว่าง

- พ่อค้า
- นายหน้า
- นักค้าอาวุธ
- สายข่าว
- กลุ่มอาชญากรรม
- ผู้มีอำนาจ

---

## Philosophy

"ทุกสิ่งมีราคา"

โลกใต้ดินเชื่อว่า

- ข้อมูลคืออำนาจ
- เงินคืออาวุธ
- ผลประโยชน์อยู่เหนือมิตรภาพ
