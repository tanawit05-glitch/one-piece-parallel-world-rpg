# Revolutionary Army

## Overview

ฐานข้อมูลกองทัพปฏิวัติในโลกคู่ขนาน

องค์กรที่ต่อต้านอำนาจของรัฐบาลโลก มีเป้าหมายเพื่อปลดปล่อยประเทศที่ถูกกดขี่ และเปลี่ยนแปลงระบบการปกครองของโลก

กองทัพปฏิวัติไม่ได้เป็นเพียงกองกำลังทหาร แต่เป็นเครือข่ายของนักปฏิวัติ สายข่าว พันธมิตร และประชาชนทั่วโลก
