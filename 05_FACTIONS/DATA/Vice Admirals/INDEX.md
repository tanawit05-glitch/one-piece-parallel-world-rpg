# Vice Admirals

## Overview

ฐานข้อมูลนายทหารระดับสูงของ Marines

Vice Admiral คือกำลังรบและผู้บัญชาการภาคสนามที่ทำหน้าที่รองจาก Admiral มีบทบาทสำคัญในการควบคุมกองเรือ ปราบโจรสลัด และรักษาความสงบของทะเล
