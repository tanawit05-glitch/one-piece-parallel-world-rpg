# PURPOSE

This document defines the simulation rules governing criminal organizations within PROJECT ASTER ENGINE.

Criminal organizations are structured groups that operate outside the law for profit, influence, power, or ideology.

Unlike pirate crews, criminal organizations usually avoid unnecessary attention and prioritize sustainable operations.

This document defines how criminal organizations grow, operate, expand, cooperate, compete, and collapse.
