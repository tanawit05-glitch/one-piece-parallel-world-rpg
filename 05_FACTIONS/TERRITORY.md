# PURPOSE

This document defines how territory is established, maintained, contested, expanded, and lost throughout PROJECT ASTER ENGINE.

Territory represents actual control over a geographical location.

Control is determined by continuous presence, influence, logistics, and operational capability.

Territory is never permanent.

# CORE PRINCIPLES

Territory is dynamic.

Control requires maintenance.

Cities

Ports

Villages

Marine Bases

Pirate Strongholds

Hidden Bases

Trade Routes

Sea Lanes

Research Facilities
