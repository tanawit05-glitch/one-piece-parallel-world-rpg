# PURPOSE

ไฟล์นี้กำหนดมาตรฐานในการสร้างเกาะใหม่ (Original Island)

เกาะใหม่ทุกเกาะต้องให้ความรู้สึกว่าเป็นส่วนหนึ่งของจักรวาล One Piece

ห้ามสร้างเกาะที่ขัดกับกฎของโลก

ทุกเกาะต้องมีเหตุผลในการดำรงอยู่

# PARALLEL WORLD RULE

PROJECT ASTER ENGINE

ใช้เกาะ Canon เป็นแกนหลัก

และเพิ่ม Original Islands เข้าไปทั่วโลก

สัดส่วนแนะนำ

- Canon Islands : 70%

- Original Islands : 30%

เกาะ Original ต้องสามารถเชื่อมต่อกับภูมิศาสตร์ของ One Piece ได้อย่างสมเหตุสมผล
