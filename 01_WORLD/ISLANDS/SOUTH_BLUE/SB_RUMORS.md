# OVERVIEW

Rumors in South Blue are a core narrative engine.

They are used by Game Masters to:
- Trigger story arcs
- Seed hidden factions
- Connect islands indirectly
- Introduce future threats
- Foreshadow world-level events

Unlike factual records, rumors may be:
- True
- Partially true
- Completely false
- Deliberately planted misinformation

# 🎭 GM USAGE SYSTEM

Each rumor has 3 layers:

## Layer 1: Surface Info
What players hear casually.

## Layer 2: Investigation Layer
What becomes true after research.

## Layer 3: Truth Layer
Actual world lore or hidden system truth.
