# PARADISE CONNECTOR SYSTEM

## 🌊 Purpose
This file connects Paradise (Grand Line First Half) with East Blue systems and defines how global world data transitions into Paradise logic.

It does NOT replace East Blue data.
It ONLY extends it.

---

## 🔗 Global Transition Rule

### From EAST BLUE → PARADISE

When a crew enters Reverse Mountain:

- All East Blue systems remain valid references
- BUT scaling rules change immediately:

| System | Change |
|---|---|
| Bounty Scale | ×10 multiplier baseline |
| Threat Evaluation | Global tier system activated |
| Navigation | Log Pose becomes mandatory |
| Marine Response | Multi-base coordination enabled |
| Pirate Behavior | Regional → Strategic |

---

## 🌐 World Integration Rule

East Blue and Paradise are NOT separate worlds.

> One world with different physics layers.
