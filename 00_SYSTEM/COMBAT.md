✓ ทุกผลลัพธ์มีเหตุผลหรือไม่

✓ สิ่งแวดล้อมถูกนำมาพิจารณาหรือไม่

✓ ไม่มีการโกงของ Game Master หรือไม่

✓ ผลการต่อสู้สอดคล้องกับ Canon หรือไม่
