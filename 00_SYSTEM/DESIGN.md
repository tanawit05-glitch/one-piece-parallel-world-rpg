# 1. PURPOSE

DICE SYSTEM คือระบบตัดสินผลลัพธ์ของเหตุการณ์ที่มีความไม่แน่นอน

หน้าที่ของระบบนี้คือ

- สร้างความยุติธรรม
- รักษาความสมจริงของโลก
- สนับสนุน Narrative Simulation
- ลดการตัดสินใจตามอำเภอใจของ AI
- ทำให้ผลลัพธ์สามารถอธิบายได้ด้วยเหตุผล

การทอยเต๋า

ไม่ใช่ระบบสุ่มเพื่อความสนุกเพียงอย่างเดียว

แต่เป็นเครื่องมือจำลอง "ความไม่แน่นอน" ภายในโลกของเกม

ทุกการทอยต้องสอดคล้องกับหลักการใน SYSTEM.md

# 6. AUTOMATIC SUCCESS

# 7. AUTOMATIC FAILURE

หากสิ่งที่พยายามทำ

โดยไม่ใช้การทอย

- มนุษย์กระโดดข้ามเกาะ

- ยิงปืนทะลุเรือรบหลายชั้นด้วยปืนธรรมดา
- ใช้ความรู้ทางการแพทย์รักษาโรคที่ไม่เคยรู้จักภายในไม่กี่วินาที

การทอยเต๋า

ไม่สามารถเปลี่ยนสิ่งที่เป็นไปไม่ได้

ให้กลายเป็นเป็นไปได้
