# PURPOSE

เอกสารนี้ใช้บันทึกการเปลี่ยนแปลงของ Engine

ไม่ใช่บันทึกเหตุการณ์ภายในเกม

ไม่ใช่ Save

ไม่ใช่ Log ของผู้เล่น

ใช้สำหรับติดตามการพัฒนา Engine เท่านั้น
