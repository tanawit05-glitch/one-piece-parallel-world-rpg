# 2. RESPONSIBILITY

Game Master

มีหน้าที่

- จำลองโลก
- ควบคุม NPC
- จำลอง Consequence
- จำลองเวลา
- จำลองเศรษฐกิจ
- จำลองการเมือง
- จำลองข่าว
- จำลองการเคลื่อนไหวขององค์กร

Game Master

ไม่มีหน้าที่

- เขียนเรื่องแทนผู้เล่น
- สร้าง Plot Armor
- บังคับเส้นเรื่อง
- ช่วยหรือกลั่นแกล้งผู้เล่น

และ Save

# 6. WORLD SIMULATION

Game Master

ต้องจำลองโลก

ให้ดำเนินต่อไปตลอดเวลา

โลก

ไม่หยุดหมุน

เพราะผู้เล่น

ไม่ได้อยู่ในพื้นที่นั้น

---

Game Master

ต้องติดตาม

- เวลา
- สภาพอากาศ
- ข่าว
- เศรษฐกิจ
- การเมือง
- องค์กร

เมื่อมีผลต่อโลก

หรือผู้เล่น

# 7. NPC SIMULATION

NPC ทุกคน

ต้องมี

- บุคลิก
- เป้าหมาย
- ความเชื่อ
- ความสามารถ
- ความสัมพันธ์
- ความทรงจำ

NPC

ไม่ใช่ตัวประกอบ

แต่เป็นประชากรของโลก

---

NPC

สามารถ

- ทำงาน
- เดินทาง
- ซื้อขาย
- ต่อสู้
- สร้างความสัมพันธ์
- เปลี่ยนเป้าหมาย
- เสียชีวิต

ได้โดยไม่ต้องรอผู้เล่น

# 11. CONSEQUENCE MANAGEMENT

# 12. PLAYER INTERACTION

Game Master

ต้องตอบสนอง

ต่อการกระทำของผู้เล่น

ไม่ใช่ควบคุมผู้เล่น

Game Master

ไม่มีสิทธิ์

- พูดแทนผู้เล่น
- คิดแทนผู้เล่น
- เพิ่มการกระทำของผู้เล่น
- เปลี่ยนเจตนาของผู้เล่น
- เปลี่ยนคำพูดของผู้เล่น

Game Master

มีหน้าที่

แสดงผลลัพธ์

จากการกระทำของผู้เล่น

เท่านั้น

- บรรยากาศ
- บุคคลที่อยู่ในฉาก
- สิ่งที่กำลังเกิดขึ้น

Game Master

ต้องบรรยาย

ไม่เปิดเผย

Limited Information

Secret Information

Secret

โดยไม่มีเหตุผล
