# 🧠 CREW MEMORY SYSTEM

Crew ไม่ได้มีแค่สถานะปัจจุบัน  
แต่มี “ความทรงจำร่วม” ที่ส่งผลต่ออนาคตทุกอย่าง

นี่คือระบบที่ทำให้ Crew “ไม่ลืมสิ่งที่เคยเกิดขึ้น”

---

# 📦 MEMORY TYPES

## 1. Shared Memories (ความทรงจำร่วม)
เหตุการณ์ที่ทั้ง Crew รับรู้ร่วมกัน

ตัวอย่าง:
- ชนะศึกใหญ่
- สูญเสียเพื่อนร่วมทีม
- พบ Devil Fruit
- ถูก Marine ไล่ล่า

ผล:
- ส่งผลต่อ Morale / Cohesion
- เปลี่ยนพฤติกรรม NPC

---

## 2. Personal Memories (ความทรงจำส่วนตัว)
เฉพาะตัวละครแต่ละคน

ตัวอย่าง:
- ความกลัว
- ความแค้น
- ความรัก
- ความลับ

ผล:
- ส่งผลต่อ Trust Network
- Trigger เหตุการณ์เฉพาะตัว

---

## 3. Trauma Memory (บาดแผล)
ความทรงจำที่ส่งผลเชิงลบระยะยาว

ผลลัพธ์:
- ลด Morale แบบถาวรบางช่วง
- สร้าง “Trigger Event”
- อาจเปลี่ยน Role หรือพฤติกรรม

---

## 4. Legacy Memory (มรดกของทีม)
เหตุการณ์ระดับตำนาน

ตัวอย่าง:
- เคยล้ม Emperors Agent
- เคยรอดจาก Buster Call
- เคยทรยศรัฐบาลโลก

ผล:
- เพิ่ม Reputation แบบก้าวกระโดด
- เปิดโลกใหม่ (New Island Events)

---

# 🔄 MEMORY PROCESS RULE

ทุก Memory จะมี:

- Timestamp (เวลาเกิด)
- Participants (ใครเกี่ยวข้อง)
- Impact Level (ระดับผลกระทบ)
- Emotion Tag (อารมณ์ของเหตุการณ์)

---

# 🧩 MEMORY TRIGGERS

Memory สามารถ “ถูกเรียกกลับมา” โดย:

- การเดินทางไปสถานที่เดิม
- พบ NPC เดิม
- เหตุการณ์คล้ายกัน
- การตัดสินใจเชิงศีลธรรม

---

# ⚠️ MEMORY DECAY (การจางหาย)

ไม่ใช่ทุก Memory จะคงอยู่เท่ากัน

- Minor Events → ค่อย ๆ จาง
- Major Events → คงอยู่ถาวร
- Trauma / Legacy → ไม่ลบ

---

# 🧠 SIMULATION RULE

Game Master ต้อง:

- ไม่ลืม Shared Memory
- ใช้ Memory เป็นเหตุผลของ NPC behavior
- สร้าง consequence ต่อเนื่อง
